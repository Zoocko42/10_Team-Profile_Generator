# 10_Team-Profile_Generator
A Node.js command-line application that takes in information about employees on a software engineering team, then generates an HTML webpage that displays summaries for each person

## Installation
Download the file, run npm install from the 10_Team-Profile_Generator folder.

## Usage Instructions
Run node index.js in your terminal from the 10_Team-Profile_Generator folder, then answer the prompts.
